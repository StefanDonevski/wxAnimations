# wxAnimations

A simple desktop animation demo built with **wxWidgets**, showcasing smooth value interpolation and real-time graph updates.

## Overview

`wxAnimations` demonstrates animated transitions and how values change over time using easing-like behavior. The app visualizes two main animated properties:

- **Horizontal Position** – shows smooth movement over time
- **Green Color Value** – demonstrates gradual color/intensity change

Both values are plotted live on graphs while the animation runs.

## Features

- Smooth animation of numeric values
- Real-time graph plotting
- Start and Reset controls
- Visual demonstration of easing-like transitions
- Simple and responsive UI built with wxWidgets
